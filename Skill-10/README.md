# Skill 10 - React Student Manager

React app demonstrating state management using `useState` with object arrays.

## Features
- Initial list with 5 students
- `useState` for:
  - `students` array
  - `newStudent` object (`id`, `name`, `course`)
- Add new student
- Delete student
- Empty-state message: `No students available`
- Basic clean CSS UI
